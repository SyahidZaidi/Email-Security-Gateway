The Project

System Overview

The Email Security Gateway Extension is an advanced Chrome extension designed to enhance the safety and usability of Gmail by providing comprehensive spam and threat detection, email encryption, and user-friendly management tools. This project is built to ensure Gmail users experience a secure environment where they can confidently send, receive, and manage their emails.


Purpose and Goals

This system was created to address the growing need for secure and reliable email management. Its primary goal is to protect users from spam, phishing attacks, and malicious content while offering seamless integration with Gmail. The system also empowers users to take control of their email security through customizable settings and real-time monitoring.


  • Spam and Threat Detection: Automatically filters spam emails and detects potentially malicious content based on user-defined keywords and domains.
  • Email Encryption: Provides a simple tool for encrypting and decrypting emails to ensure the confidentiality of sensitive information.
  • Customizable Security Settings: Allows users to input custom spam and threat keywords and toggle specific security measures.
  • Real-Time Dashboard: Displays incoming filtered emails in a visually appealing, interactive table format.
  • User-Friendly Interface: Designed to be easy to navigate with intuitive options and a modern, dark-themed layout.
