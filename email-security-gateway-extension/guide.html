User Guide

User Guide for Email Security Gateway Extension

Welcome to the Email Security Gateway Extension User Guide. This document will help you understand how to use the features of the extension, set up your preferences, and troubleshoot common issues.

Getting Started

Installation

To install the Email Security Gateway Extension, follow these steps:

1. Go to the Chrome Web Store and search for "Email Security Gateway Extension.

2. Click on Add to Chrome.

3. Confirm the installation by selecting Add Extension.

4. The extension icon will appear in your browser's toolbar once installed.


Getting Started

Once the extension is installed:

1. Click on the extension icon in your Chrome toolbar.

2. Sign in using your Google account or ensure that the extension is connected to your Gmail account for access.

3. Access the extension's main dashboard or settings page.


Features Overview

  • Spam Detection: Filters spam emails based on user-defined keywords.
  • Threat Detection: Identifies and highlights potentially malicious emails based on keywords and domain checks.
  • Email Encryption: Provides a simple tool for encrypting sensitive email messages.
  • Email Decryption: Allows users to decrypt messages using a secret key.
  • Customizable Dashboard: Displays filtered emails and real-time notifications.

Using the Extension

Email Filtering

1. Open the extension popup by clicking the extension icon.

2. Ensure that the Spam Detection or Threat Detection checkboxes are selected on the settings page.

3. The system will automatically filter and highlight spam or threat emails.

4. View detected emails in the Filtered Emails section of the extension's popup or dashboard.


Email Encryption

1. Go to the Encryption tab in the extension or navigate to the encryption page.

2. Enter the message you wish to encrypt in the text area labeled Enter your message here....

3. Click Encrypt Message.

4. The encrypted message will appear in the Encrypted Message box. You can copy and paste this encrypted text for safe sharing.


Email Decryption

1. Navigate to the Decryption tab on encryption page

2. Paste the encrypted message into the Enter the encrypted message here... text area.

3. Enter the corresponding secret key.

4. Click Decrypt Message to reveal the original content in the Decrypted Message box.



Settings Page Overview

Access the settings page by:

1. Clicking the Go to Settings button in the extension popup.

2. Navigating to settings.html.


Adding/Managing Spam Keywords

1. Locate the Spam Keywords section.

2. Type a keyword in the input box and click Add to include it in the detection list.

3. View, edit, or delete keywords from the list below the input box.


Adding/Managing Threat Keywords and Domains

1. Go to the Threat Detection section.

2. Add keywords or domains in their respective input fields and click Add.

3. Manage existing entries using the provided options.


Dashboard and Notifications

1. The Dashboard displays filtered emails in a table format.

3. The dashboard updates in real-time with incoming filtered emails.


Troubleshooting

Issue: The extension isn't filtering emails.

1. Ensure that the extension has the necessary permissions to access your Gmail account.

2. Verify that the filtering checkboxes are checked on the settings page.


Issue: Encrypted messages won't decrypt properly.

1. Ensure that you are using the correct secret key.

2. Double-check for any accidental modifications to the encrypted text.
